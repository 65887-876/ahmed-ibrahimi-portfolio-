# Ahmed Ibrahimi Portfolio

A creative portfolio website built as a support project for a childhood friend starting and growing his photography business.

This project was made with heart: a friend helping a friend turn talent into an online presence that feels professional, cinematic, and personal.

## Story

Ahmed Ibrahimi is a photographer, not a programmer.  
This site was crafted to give him:

- a clean homepage with strong visual identity
- a professional gallery experience
- a linktree-style page for quick social and booking access

## Live Structure

- `/` -> Creative home landing page
- `/gallery` -> Signature gallery with filters and full-screen viewer
- `/linktree` -> Social links hub (Instagram, Email, WhatsApp, Telegram)

## Features

- Cinematic dark visual theme
- Mobile-first responsive design
- Category-filtered portfolio gallery
- Full-screen image viewer with keyboard navigation
- Route-aware scroll-to-top behavior
- Social-first contact flow for real clients

## Tech Stack

- React 19 + Vite 7
- Tailwind CSS v4
- React Router
- Lucide React icons

## Local Development

```bash
npm install
npm run dev
```

Runs on:

- `http://localhost:5173`
