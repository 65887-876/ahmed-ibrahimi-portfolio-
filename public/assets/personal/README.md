# Personal Page Images

Place your images in this folder with these exact names:

## Naming Guide

| File Name       | Your Photo                          |
|-----------------|-------------------------------------|
| `portrait.jpg`  | Your normal/regular photo            |
| `desert-1.jpg`  | Desert photo 1                       |
| `desert-2.jpg`  | Desert photo 2                       |
| `desert-3.jpg`  | Desert photo 3                       |
| `food-1.jpg`    | You with food you love (photo 1)     |
| `food-2.jpg`    | You with food you love (photo 2)     |
| `macbook.jpg`   | You with your MacBook                |
| `city.jpg`      | You in a big city                    |

## Where They Appear

- **Who I Am** → `portrait.jpg`
- **The Curiosity** (travel) → `desert-1.jpg`, `desert-2.jpg`, `desert-3.jpg`, `city.jpg`
- **The Taste** (food) → `food-1.jpg`, `food-2.jpg`
- **The Connection** (work) → `macbook.jpg`

## Supported Formats

.jpg, .jpeg, .png, .webp
